Simon Game

Simon is a classic memory game where players need to repeat a sequence of colors and sounds in the correct order. This project brings the iconic Simon game to your browser.
Description

Simon is a game of memory skill. The game consists of four colored buttons, each producing a particular tone when it is pressed or activated by the device. A round in the game consists of the device lighting up one or more buttons in a random order, after which the player must reproduce that order by pressing the buttons. As the game progresses, the number of buttons to be pressed increases.
Features

    Simple and intuitive interface.
    Engaging gameplay that tests your memory skills.
    Interactive feedback with sounds and visual effects.
    Responsive design suitable for desktop and mobile devices.

How to Play

    1.Open the index.html file in your web browser.
    2.Press any key to start the game.
    3.Watch and listen as the game lights up buttons in a sequence.
    3.Repeat the sequence by clicking on the buttons in the same order.
    4.As you progress, the game will increase the length of the sequence,
    5.challenging your memory skills.
    6.If you make a mistake, the game will end, and you can restart by pressing any key again.
